# Vaulty JS SDK

It is mono repository for Vaulty JS SDK and its playground.
For controlling that, we use [yarn workspaces](https://yarnpkg.com/lang/en/docs/workspaces/).

## Technical requirements

 - Node v10.13+
 - Yarn 1.19+
 
## Initialize mono repository
 
```
yarn install
```
