# JS SDK elements (fields and its controllers)

It creates content inside of iframe by get params (for example, an input)

## Technical requirements

 - Node v10.13+
 - Yarn 1.19+

## Initialize repository

```sh
yarn install
```

## Start Dev version of elements controller

```sh
yarn start
```

## Run linting

```sh
yarn lint
```

## Run tests

```sh
yarn test
```

## Build library

```sh
yarn build
```
